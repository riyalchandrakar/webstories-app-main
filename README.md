# Web Stories App 📱

A full-stack web stories application similar to Instagram/Google Web Stories with CMS and Player.

![Web Stories](https://img.shields.io/badge/Web-Stories-blue)
![React](https://img.shields.io/badge/React-18.2.0-61dafb)
![Node.js](https://img.shields.io/badge/Node.js-Express-green)
![MongoDB](https://img.shields.io/badge/MongoDB-Atlas-green)

## ✨ Features

- **📝 CMS Dashboard** - Create, edit, delete stories
- **🎬 Story Player** - Instagram-like story viewing experience
- **🖼️ Multiple Media** - Support for images and videos
- **🎭 Animations** - Fade, slide, zoom transitions
- **📂 Categories** - Organize stories by categories
- **📱 Responsive** - Works perfectly on desktop and mobile
- **⚡ Real-time** - Auto-advance stories with progress bars
